## Drivespace Storage Boundary (Decision Record)

Drivespace is **storage only**. It handles:
- File upload & versioning
- Folder management with visibility controls (private / role / members / public)
- File sharing (user/role permissions)
- File download

It does NOT handle:
- Document composition (rich text editor / WYSIWYG writing)
- Export to PDF/DOCX/HTML
- Status workflow (draft → pending_review → approved → published)
- Publishing to articles, events, social, or mailing lists

If a document lifecycle module is needed in future, it will be a separate module (e.g. `documents`) with Drivespace as its storage backend.
