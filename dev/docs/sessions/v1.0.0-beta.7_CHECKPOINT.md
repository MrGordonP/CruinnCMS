## What Changed

### Repo restructure
- Engine source moved to `CruinnCMS/` subdirectory; web root moved to `public_html/`
- Layout mirrors live cPanel hosting: `CruinnCMS/` → `/home/user/`, `public_html/` → `public_html/`
- `.gitignore` updated for new paths; `CruinnCMS/instance/*/` excluded (instance data, mail archives, forum backups)

### Settings media browser (`templates/admin/settings/site.php`)
- Logo and Banner fields replaced with Browse… button + preview image
- Inline media picker modal: grid view, double-click or Select to insert, Upload new… for direct upload
- `.media-input-row` CSS added to `admin-acp.css`

### Nav menu CSS fix (`public_html/css/style.css`)
- `.nav-list` now inherits `justify-content` from parent `<nav>` block
- Fixes alignment controls (`flex-start` / `flex-end`) from properties panel propagating into menu items

### Blog article body rendering (`templates/public/articles/show.php`)
- Template was using old `$blocks` loop / `$article['body']` — neither existed
- Fixed to output `$body_html` (result of `renderArticleBlocks()`)

### WP image URL rewrite
- All `https://geology.ie/wp-content/uploads/` URLs in `article_blocks.inner_html` rewritten to `/storage/iga/media/`
- 19 rows updated; WP uploads archive copied to `public_html/storage/iga/media/`

### Nav menu block type — dropdown support (`src/BlockTypes/nav-menu/definition.php`)
- Renderer previously only fetched root-level items (`parent_id IS NULL`)
- Rewrote to fetch all menu items in one query, index by id, build full recursive dropdown tree
- Outputs `<ul class="nav-dropdown">` child lists; `open_new_tab` respected

### Main navigation rebuilt (DB — menu id=1)
- Full geology.ie hierarchy imported via SQL
- 7 top-level items: Home, Events, Blog, Forum, Archive and Resources, Membership and More, About
- 3 dropdown groups with 4–6 children each
- Internal routes for: Home, Events, Blog, Forum, Council, Membership
- External geology.ie URLs for archive/resources pages not yet migrated
