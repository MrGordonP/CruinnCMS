### Shared PanelCollapse module (`ccea592`, `41db16a`, `a6b00bf`, `c8c74d5`)

**Problem:** Left/right panel collapse logic was duplicated in `source-editor.js` only, and not present at all on admin three-pane pages. Centre panel of the source editor had no collapse at all.

**Solution:** Extracted a single shared `PanelCollapse` IIFE module. All three-pane layouts (platform source editor + all admin panel-layout pages) delegate to it.

#### New file
- `public_html/js/admin/panel-collapse.js` — `PanelCollapse.init(configs)` + `PanelCollapse.expand(panelId)`
  - Config per panel: `{ panelId, toggleId, storeKey, side }`
  - Icons: left `▶/◀`, centre `▶/◀` (collapses left), right `◀/▶`
  - Persists to `localStorage` via `storeKey`

#### CSS
- `public_html/css/admin-panel-layout.css` — `.pl-sidebar.collapsed` / `.pl-detail.collapsed` collapse to 32px strips; CSS `:has()` overrides grid columns automatically
- `public_html/css/platform.css` — `.source-code-pane.collapsed` collapses to 32px; toggle button absolutely positioned on left edge, always visible

#### Admin templates (7 files)
All received:
- `id="pl-sidebar"` on `.pl-sidebar` div
- `<button class="pl-panel-toggle" id="pl-sidebar-toggle">` in `.pl-sidebar-header`
- `<button class="pl-panel-toggle" id="pl-detail-toggle">` in `.pl-detail-header`

Files: `groups/index.php`, `roles/index.php`, `roles/edit.php`, `pages/index.php`, `media/index.php`, `menus/index.php`, `site-builder/structure.php`

#### Admin JS (6 files)
`PanelCollapse.init([...])` call appended outside each page IIFE:
`groups.js`, `roles.js`, `pages.js`, `media.js`, `menus.js`, `structure.js`

#### Platform source editor
- `CruinnCMS/templates/platform/source-editor.php` — `id="source-code-pane"` on centre pane; toggle button moved to be a direct child of `.source-code-pane` (above the form, so it stays visible when form is hidden on collapse)
- `public_html/js/platform/source-editor.js` — own panel logic removed; delegates to `PanelCollapse.init`; `panel-collapse.js` loaded via explicit `<script>` tag before `source-editor.js`

#### `CruinnCMS/templates/admin/layout.php`
`panel-collapse.js` added to `$adminModules` array (always-loaded on every admin page).
