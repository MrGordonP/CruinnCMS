### 2. Template blank canvas fix (committed `e2330e6`)

**Problem:** After migration 012, template canvas blocks live in `pages` / `pages_draft` with
`template_id = N, page_id = NULL`. All block-writing action methods in `CruinnController`
used `WHERE page_id = ?`, returning nothing or writing to the wrong column.

**Fix:** Added `resolveBlockKey(int $pageId): array` private helper method:
```php
private function resolveBlockKey(int $pageId): array
{
    $tplId = (int) ($this->db->fetchColumn(
        'SELECT id FROM page_templates WHERE canvas_page_id = ? LIMIT 1', [$pageId]
    ) ?: 0);
    return $tplId > 0
        ? ['col' => 'template_id', 'val' => $tplId]
        : ['col' => 'page_id',     'val' => $pageId];
}
```

Updated all six action methods to call `$bk = $this->resolveBlockKey($pageId)` and use
`{$bk['col']} = ?` / `$bk['val']` in all block queries:
- `recordAction()` — draft INSERT + seq cap queries
- `undo()` — MAX seq SELECT, DELETE, published fallback SELECT
- `publish()` (block mode) — DELETE published, INSERT SELECT (both `page_id` + `template_id` columns copied), DELETE draft
- `discardDraft()` — DELETE draft
- `reloadFromSource()` — DELETE draft

html/file render mode paths in `publish()` left unchanged — template canvases never use those modes.

**File modified:** `src/Controllers/CruinnController.php`

### 3. Zones management UI (committed `159ea80`, `3ae914b`, `b7aa4e7`)

Added a **Zones** tab to the Site Builder (between Templates and Menus).

**New route:** `GET /admin/site-builder/zones` → `builderZones()`  
**New route:** `POST /admin/site-builder/zones/new` → `builderNewZone()`

**`builderZones()`** — lists all `pages_index WHERE canvas_type='zone'` with title, zone_name, status, updated date, and Edit Canvas link (→ `/admin/editor/{id}/edit`).

**`builderNewZone()`** — validates title + zone_name (lowercase slug pattern), checks for duplicate zone_name, creates `pages_index` row with `canvas_type='zone'`, redirects directly to editor.

**Files modified/created:**
- `config/routes.php` — two new routes
- `src/Admin/Controllers/SiteBuilderController.php` — `builderZones()` + `builderNewZone()`
- `templates/admin/site-builder/_tabs.php` — Zones tab added
- `templates/admin/site-builder/zones.php` — new template (list + create form)

**Lessons learned (bugs fixed on live):**
- CSRF token: correct pattern is `<input type="hidden" name="csrf_token" value="<?= \Cruinn\CSRF::getToken() ?>">` — no `CSRF::field()` method exists
- Escape: use global `e()` helper in templates — `$this->escape()` does not exist on Template
