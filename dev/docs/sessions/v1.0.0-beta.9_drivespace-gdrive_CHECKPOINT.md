## Key design decisions

- **Virtual root (no folder configured):** `listFolder(null)` issues `sharedWithMe=true` query ‚Äî all 20+ top-level folders shared with the service account appear, fully navigable. No need to share the Drive root.
- **Pinned root (folder ID configured):** Restricts the browser to that subtree only.
- **Shared Drive (Google Workspace):** Optional `gdrive.shared_drive_id`; uses `corpora=drive` API param.
- **No Composer dependency added** ‚Äî JWT signing done with `openssl_sign()` / `base64_encode()` in pure PHP.
- **Settings upsert** ‚Äî all settings saves safe to run whether or not the migration has been applied.
